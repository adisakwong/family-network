# Family Network - GitHub Pages + Google Apps Script

โปรเจกต์นี้เป็นการนำมารีแฟกทอริ่ง (Refactoring) ให้ฝั่งหน้าต่างผู้ใช้งาน (Frontend) ทำงานแบบ Static Website ซึ่งสามารถนำไปอัปโหลด (Deploy) บนบริการโฮสต์ได้ฟรีอย่าง **GitHub Pages** ในขณะที่ระบบฐานข้อมูลและการจัดการไฟล์จะยังคงรันผ่าน **Google Apps Script** เป็นเหมือน API Server (Backend)

## วิธีการติดตั้งและใช้งาน (Deployment Guide)

### ส่วนที่ 1: การติดตั้ง Backend (Google Apps Script)
1. ไปที่ [Google Sheets](https://docs.google.com/spreadsheets/) สร้างหน้าใหม่และตั้งชื่อตามต้องการ (เช่น "Family Network DB")
2. คลิกที่เมนู `ส่วนขยาย` (Extensions) > `Apps Script`
3. ในหน้าต่าง Apps Script คัดลอกโค้ดทั้งหมดจากไฟล์ `Code.gs` ที่อยู่ในโปรเจกต์นี้ ไปวางทับใน `Code.gs` บน Apps Script แล้วกดบันทึก
4. กดยืนยันการเผยแพร่:
   - มุมขวาบนคลิกปุ่ม **"การทำให้ใช้งานได้"** (Deploy) > **"การทำให้ใช้งานได้รายการใหม่"** (New deployment)
   - ตรงช่องซ้ายมือ "เลือกประเภท" ให้เลือกรูปฟันเฟือง แล้วติ๊ก **"เว็บแอป"** (Web app)
   - ตั้งค่า:
     - **คำอธิบาย:** "Family Network API" (พิมพ์อะไรก็ได้)
     - **เรียกใช้ในฐานะ:** เลือก **"ฉัน"** (Me)
     - **ผู้มีสิทธิ์เข้าถึง:** เลือก **"ทุกคน"** (Anyone)
   - กดปุ่ม **การทำให้ใช้งานได้** (Deploy)
5. **(สำคัญมาก)** หากเป็นการรันครั้งแรก ระบบจะขอสิทธิ์เข้าถึง:
   - คลิก **"ตรวจสอบสิทธิ์"** (Authorize access) แล้วเลือกบัญชี Google ของคุณ
   - กดขั้นสูง (Advanced) และเลือกไปยังสคริปต์ (Go to ... UNSAFE)
   - กดยอมรับการเข้าถึง (Allow) เพื่อให้สามารถบันทึกและอ่านข้อมูลจาก Google Sheets / Google Drive
6. ให้คัดลอก **URL ของเว็บแอป** ที่ลงท้ายด้วย `/exec` มาเก็บไว้ (จะนำไปใช้ในหน้าเว็บ)

### ส่วนที่ 2: การพัฒนาส่วนหน้าจอ (Frontend) บน GitHub Pages
เมื่อเราได้ URL ของ Web App API มาจากส่วนบนแล้ว ขั้นตอนต่อไปคือการอัปโหลดไฟล์เว็บขึ้น GitHub

1. เปิดไฟล์ `index.html` ออกมา (คุณสามารถเปิดด้วย Notepad, VS Code หรือตัวแก้ไขข้อความใด ๆ)
2. เลื่อนลงมาล่างสุด (ก่อนจบ `</body>`) จะเจอคำสั่งต่อไปนี้:
   ```javascript
   // ใส่ URL Web App ของ Google Apps Script ของคุณที่นี่
   const GAS_WEB_APP_URL = 'YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL';
   ```
3. นำ URL ที่คัดลอกมาจาก Apps Script เข้ามาแทนคำว่า `YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL` ให้เรียบร้อย
4. นำไฟล์ทั้งหมด **ยกเว้น** `Code.gs` (`index.html`, `style.css`, `script.js`, และ `README.md`) ไปใส่ใน Repository บน GitHub ของคุณ
5. ไปที่แท็บ **Settings** ในหน้าของ Repository บน GitHub
6. หาคำว่า **Pages** แถบด้านซ้าย
7. ในหน้า GitHub Pages ภายใต้ **Build and deployment** > Source เลือกเป็น **Deploy from a branch** 
8. เลือกสาขา `main` หรือ `master` ในเลือกโฟลเดอร์ `/ (root)` และคลิก Save
9. รอสักครู่ GitHub Pages จะสร้างลิงก์สำหรับเข้าเว็บไซต์ให้ เมื่อขึ้นแถบสีเขียวที่มี URL ก็สามารถกดเข้าไปใช้งานแอปพลิเคชันรูปแบบ Website คุณภาพสูงได้เลย!

### ฟีเจอร์ที่น่าสนใจ
- ระบบฐานข้อมูลผ่าน Google Sheets 
- รองรับการกล้องด้วยคำสั่ง `getUserMedia` ผู้ใช้งานสามารถใช้มือถือสลับกล้องหน้า-หลังเพื่อถ่ายรูปยืนยันตัวตนได้ทันที
- ประมวลผลและบีบอัดรูปให้เป็น Base64 แล้วโยนขึ้นไปจัดเก็บเป็นไฟล์อยู่ในระบบ Google Drive ของผู้ออกอัตโนมัติ 
- ไม่ต้องเช่า Server ค่าบริการ 0 บาท!
